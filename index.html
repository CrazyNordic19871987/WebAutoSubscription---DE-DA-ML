Выводы

Все три гипотезы отклонены - во всех случаях есть статистически значимые различия в конверсии между группами. Однако размер эффекта (Cramér's V) во всех случаях малый, что говорит о том, что практическая значимость различий невелика.

Интерпретация

Наиболее важные факторы для предсказания конверсии:

  1. Источник трафика - разные каналы привлечения дают разную конверсию
  2. Тип трафика - органический трафик конвертируется лучше
  3. Номер визита - повторные посетители могут быть более заинтересованы
  4. География - жители крупных городов чаще оставляют заявки

5. Data Engineer - База данных и пайплайн

База данных

Используется SQLite для хранения и запросов к данным.

Схема БД

-- Таблица сессий CREATE TABLE ga_sessions ( session_id TEXT PRIMARY KEY, client_id REAL, visit_date TEXT, visit_time TEXT, visit_number INTEGER, utm_source TEXT, utm_medium TEXT, utm_campaign TEXT, device_category TEXT, geo_country TEXT, geo_city TEXT, created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ); -- Таблица хитов CREATE TABLE ga_hits ( id INTEGER PRIMARY KEY AUTOINCREMENT, session_id TEXT, hit_date TEXT, hit_time REAL, hit_number INTEGER, event_action TEXT, FOREIGN KEY (session_id) REFERENCES ga_sessions(session_id) ); -- Индексы CREATE INDEX idx_hits_session ON ga_hits(session_id); CREATE INDEX idx_sessions_date ON ga_sessions(visit_date); CREATE INDEX idx_sessions_source ON ga_sessions(utm_source);

Пайплайн для новых данных

Пайпайн обрабатывает JSON файлы:

  1. Читает JSON из папки new_data/
  2. Валидирует и преобразует данные
  3. Загружает в SQLite БД
  4. Перемещает обработанный файл в archive/

Пример JSON

{ "sessions": [ { "session_id": "test_001", "client_id": 123456.0, "visit_date": "2022-01-15", "visit_time": "14:30:00", "visit_number": 1, "utm_source": "test_source", "utm_medium": "organic", "device_category": "desktop", "geo_country": "Russia", "geo_city": "Moscow" } ], "hits": [ { "session_id": "test_001", "hit_date": "2022-01-15", "hit_time": 14.5, "hit_number": 1, "event_action": "sub_car_claim_click" } ] }

6. Программный код

Data Analyst - Основной скрипт

""" СберАвтоподписка - Data Analyst Основной скрипт анализа """ import pandas as pd import numpy as np import matplotlib.pyplot as plt import seaborn as sns from scipy import stats import warnings import os # Константы TARGET_ACTIONS = [ 'sub_car_claim_click', 'sub_car_claim_submit_click', 'sub_open_dialog_click', 'sub_custom_question_submit_click', 'sub_call_number_click', 'sub_callback_submit_click', 'sub_submit_success', 'sub_car_request_submit_click' ] ORGANIC_MEDIUMS = ['organic', 'referral', '(none)'] # Загрузка данных sessions = pd.read_csv('ga_sessions.csv', low_memory=False) hits = pd.read_csv('ga_hits.csv', usecols=['session_id', 'event_action']) # Определение целевых действий hits['is_target'] = hits['event_action'].isin(TARGET_ACTIONS) target_sessions = hits.groupby('session_id')['is_target'].any() data = sessions.merge(target_sessions.reset_index(), on='session_id', how='left') data['has_target_action'] = data['is_target'].fillna(False) # Расчет CR CR = data['has_target_action'].mean() print(f"Общий CR: {CR*100:.2f}%") # Проверка гипотез def test_hypothesis(data, group_col, name1, name2, alpha=0.05): g1 = data[data[group_col]==True]['has_target_action'] g2 = data[data[group_col]==False]['has_target_action'] table = np.array([[g1.sum(), len(g1)-g1.sum()], [g2.sum(), len(g2)-g2.sum()]]) chi2, p, dof, exp = stats.chi2_contingency(table) return chi2, p # Тестирование data['is_organic'] = data['utm_medium'].isin(ORGANIC_MEDIUMS) chi2, p = test_hypothesis(data, 'is_organic', 'Органический', 'Платный') print(f"Chi-square: {chi2:.2f}, p-value: {p:.6f}")

ML - Обучение модели

from sklearn.model_selection import train_test_split from sklearn.ensemble import RandomForestClassifier from sklearn.metrics import roc_auc_score import pickle # Фичи feature_cols = ['visit_number', 'visit_hour', 'visit_dayofweek', 'is_organic', 'is_mobile', 'device_category_enc', 'utm_source_enc', 'geo_city_enc'] X = data[feature_cols].fillna(0) y = data['target'] X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2) # Обучение model = RandomForestClassifier(n_estimators=100, max_depth=8, class_weight='balanced') model.fit(X_train, y_train) pred = model.predict_proba(X_test)[:, 1] auc = roc_auc_score(y_test, pred) print(f"ROC-AUC: {auc:.4f}") # Сохранение with open('model.pkl', 'wb') as f: pickle.dump({'model': model, 'features': feature_cols}, f)

DE - Пайплайн

import sqlite3 import json import os DB_PATH = 'sberautopodpiska.db' NEW_DATA_PATH = 'new_data' def process_json(file_path): with open(file_path, 'r') as f: data = json.load(f) conn = sqlite3.connect(DB_PATH) # Загрузка сессий for s in data.get('sessions', []): cols = ', '.join(s.keys()) vals = list(s.values()) cursor.execute(f"INSERT OR IGNORE INTO ga_sessions ({cols}) VALUES (?,...)", vals) # Загрузка хитов for h in data.get('hits', []): cols = ', '.join(h.keys()) vals = list(h.values()) cursor.execute(f"INSERT INTO ga_hits ({cols}) VALUES (?,...)", vals) conn.commit() conn.close() # Запуск files = [f for f in os.listdir(NEW_DATA_PATH) if f.endswith('.json')] for f in files: process_json(f"{NEW_DATA_PATH}/{f}")
