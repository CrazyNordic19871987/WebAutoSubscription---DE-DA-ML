# WebAutoSubscription---DE-DA-ML
 Финальная работа по курсу "Введение в Data Science". Анализ данных сайта — сервиса долгосрочной аренды автомобилей для физических лиц.  Проект включает три специализации: - Data Analyst — разведочный анализ, проверка гипотез, бизнес-вопросы - ML Engineer — предиктивная модель, API сервис - Data Engineer — база данных, пайплайн обработки данных
